# Ocean Taste 🌊🐟

Ocean Taste is a modern seafood restaurant website built using HTML and CSS.  
The website features a clean ocean-inspired design with blue gradients, black borders, responsive layouts, and sections for showcasing seafood dishes and restaurant information.

---

## Features

- Modern seafood-themed UI
- Responsive design for mobile and desktop
- Ocean blue color palette
- Smooth hover effects
- Hero section with image placement
- About section
- Navigation bar
- Footer section
- Easy to customize

---

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)
